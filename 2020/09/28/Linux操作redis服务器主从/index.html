什么是redis主从复制

​ redis持久化保证了redis服务重启也不会丢失数据，因为redis服务重启后会将硬盘上持久化的数据恢复到内存中，但是当redis服务器的硬盘损坏了可能会导致数据丢失，如果通过redis的主从复制机制就可以避免这种单点故障。主机数据更新后根据配置和策略，自动同步到备机。

主从复制能做什么？

  1. 读写分离
  2. 容灾恢复
