支付宝沙箱支付操作
发表于2020-09-28|更新于2020-09-30|三方插件
|阅读量:

  1. 进入支付宝开放平台

  2. 进入沙箱应用，找到RSA2(SHA256)密钥进行配置

  3. 下载支付宝开放平台开发助手，进行生成秘钥

  4. 这时会生成两个秘钥，一个应用公钥，一个应用私钥

  5. 将应用公钥进行复制，到RSA2(SHA256)密钥将应用公钥粘贴进去，这时会自动生成一个支付宝公钥

  6. 到这一步我们会得到三个秘钥，分别是：支付宝公钥、应用公钥、应用私钥(在支付宝开放平台开发助手里)

  7. 拿着这三个秘钥在Django项目下的支付app根目录创建文件夹keys

  8. 在你的沙箱应用里配置好授权回调地址(我这里写的回调到前端 “http://127.0.0.1:8080/payment/" 这个页面中)

  9. 开始写配置settings.py

  1. 安装Alipay(pip install alipay)

  2. myapp/utils.py (编写有关支付方法)

  3. 开始写视图(myapp/views.py)
